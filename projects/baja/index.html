wireless data transmission

2011

Installed on University of Rochester’s SAE Baja Team for 2012 competition.

The Arudino Fio is used in this project to sample analog data signals and send them wirelessly using a pair of Xbee modules. All data is displayed on the dashboard using analog electronics and in realtime on a nearby computer that is running the display program.

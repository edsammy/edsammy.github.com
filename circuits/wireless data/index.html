wireless data transmission

The Arudino Fio is used in this project to sample analog data signals and send them wirelessly using a pair of Xbee modules. All data is displayed in realtime on a nearby computer that is running the display program. This device was placed on a vehicle to sample speed, tach, engine temperature, and fuel level. A simple communication system can also be used to transmit short messages between the driver and the individual monitoring the data.
